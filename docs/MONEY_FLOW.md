## 2. 口座・カード構成（入居者1名あたり）

| 区分 | 名義 | 口座/カード | 用途 |
|------|------|-----------|------|
| 本体口座 | 入居者 | 既存の銀行口座 | 年金受取・資産保全 |
| 生活費口座 | 入居者 | 住信SBI 普通預金 | 月次生活費の管理 |
| 物理デビット | 入居者 | 住信SBI Mastercard デビット | 対面決済 |
| バーチャルデビット | 入居者 | 住信SBI スマホデビット | オンライン決済（Phase 2） |

## 3. 各チャネルの詳細

### ① 口座振替（自動）
- 家賃・共益費・食費など固定費
- 入居時に口座振替を設定、以後は自動
- RSWは一切関与しない
- システムはCSV取込で結果を記録

### ② デビットカード（物理）
- 日用品・消耗品の対面購入
- カードは施設内のセキュリティボックスに保管
- RSWがチェックアウト → 買い物 → レシート撮影 → チェックイン
- 1回の利用上限は任意後見契約で設定（○○円）
- システムは貸出ログ + 利用通知を記録

### ③ システム決済（Phase 2）
- オンライン購入やChoice100サービスの手配
- RSWがアプリ上で購入申請を作成
- admin or supervisorが承認
- システムが暗号化されたカード番号で決済実行
- RSWはカード番号を知らない設計

## 4. 法的基盤

- **任意後見契約**（公正証書）でRSWに限定的代理権を付与
- 代理権の範囲に「生活費専用口座の管理」「デビットカードの利用」を明記
- **後見監督人**が月次で監査（月次レポートPDFを確認）
- 定額振替の金額変更・都度振替は、アレソノワ本部 + 後見監督人の双方承認が必要

## 5. 不正防止の仕組み

| リスク | 対策 | Phase |
|--------|------|-------|
| 金銭横領 | 本体口座への直接アクセス不可。生活費口座のみ、上限付き | P0 |
| カード不正利用 | チェックアウト/チェックイン制。持出ログ記録 | P1 |
| レシート偽造 | 24h以内のレシート添付義務。未添付アラート | P1 |
| 高額取引 | 1回上限の設定。超過は決済不可（デビットの銀行設定） | P1 |
| 残高着服 | CSV取込で残高をシステム管理。家族/後見監督人が確認可能 | P0 |
| キックバック | 事業者選定の可視化（Phase 2のサービスカタログ） | P2 |
